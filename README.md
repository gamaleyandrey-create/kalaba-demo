# Kalaba Demo

Демо-прототип Kalaba для GitHub Pages.

- Русский язык по умолчанию
- Регион: Испания
- Переключение языков: русский, украинский, испанский, английский
- Фото-блоки на главной странице
- Активные кнопки в предложениях: смотреть, чат, принять
- Все файлы в корне, можно загружать поверх старой версии
