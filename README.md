# Kalaba Complete Clean

Полный чистовой пакет Kalaba для GitHub Pages.

- Регион использования: Испания
- Языки: Русский, Українська, Español, English
- Русский по умолчанию
- Испанские города, адреса и телефоны
- Карта Испании
- SVG-фото и визуальные блоки
- Все файлы в корне, без папок
- Можно загрузить поверх старых файлов
