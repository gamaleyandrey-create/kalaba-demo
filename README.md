# Kalaba

Демонстрационный прототип Kalaba для GitHub Pages.

## Особенности

- Название без RU в интерфейсе и файле проекта
- Русский язык по умолчанию
- Переключатель языков: Русский / Українська
- Все файлы в корне репозитория
- Совместимо с GitHub Pages
- 47 HTML-страниц
- styles.css
- app.js

## Публикация

Загрузите все файлы в корень репозитория GitHub Pages.
