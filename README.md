# Kalaba

Чистовой HTML-прототип Kalaba для GitHub Pages.

## Что внутри

- Единый стиль для всех страниц
- Русский язык по умолчанию
- Украинский язык в переключателе
- Светлая и тёмная темы
- Все файлы без папок
- Файлы с именами старых версий, чтобы можно было загрузить поверх без ручного удаления

## Публикация

Загрузите все файлы в корень репозитория GitHub Pages.
